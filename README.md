### 주요 기능
  - 시작,종료 페이지 로딩 애니메이션 구현
- Redux 활용 상태 관리
    - 바탕 화면 아이콘, 창 드래그&드롭  위치 관리
    - 창 최대화, 최소화, 활성화 관리
    - 투두리스트 작성,수정,삭제 기능 (로컬스토리지)
- 재사용 컴포넌트
    - 창 컴포넌트 > iframe활용 배포 프로젝트 띄우기, 아이콘 타입 별 크기
    - 아이콘 컴포넌트  > 창 메뉴 아이콘, 시작 메뉴 아이콘
    - 뱃지 컴포넌트 > 리스트 생성, 수정, 상태 뱃지
